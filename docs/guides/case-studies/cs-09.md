# CS-09 User Guide — Insurance Claims

**Tier:** 3 · **ID:** `cs-09`

## Problem
Claims adjusters manually verify coverage and docs.

## Demo story
Water damage claim with complete documentation within coverage.

## How to run (no code)

1. Open **Case Study Gallery** in the investor console
2. Select **Insurance Claims**
3. Choose a scenario:
   - **Approved claim** (`default`) → expected: `ALLOW`
   - **Denied — fraud flag** (`denied`) → expected: `DENY`
4. Click **Run demo** — view outcome, proof, and data sources

## API

```bash
curl -X POST http://localhost:8000/case-studies/cs-09/run \
  -H "Content-Type: application/json" \
  -d '{"scenario": "default"}'
```
