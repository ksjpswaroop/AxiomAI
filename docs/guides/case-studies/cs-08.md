# CS-08 User Guide — Contract / SLA Analysis

**Tier:** 3 · **ID:** `cs-08`

## Problem
Manual contract review misses SLA breach triggers.

## Demo story
Uptime fell to 99.2% vs 99.9% SLA; renewal notice window opens.

## How to run (no code)

1. Open **Case Study Gallery** in the investor console
2. Select **Contract / SLA Analysis**
3. Choose a scenario:
   - **SLA breach triggered** (`default`) → expected: `OBLIGATION_TRIGGERED`
   - **Renewal notice required** (`renewal`) → expected: `OBLIGATION_TRIGGERED`
4. Click **Run demo** — view outcome, proof, and data sources

## API

```bash
curl -X POST http://localhost:8000/case-studies/cs-08/run \
  -H "Content-Type: application/json" \
  -d '{"scenario": "default"}'
```
