# CS-05 User Guide — AI Code Review Guardrail

**Tier:** 5 · **ID:** `cs-05`

## Problem
LLM-generated code may include secrets or insecure routes.

## Demo story
Deploy pipeline detected hardcoded API key and unauthenticated admin route.

## How to run (no code)

1. Open **Case Study Gallery** in the investor console
2. Select **AI Code Review Guardrail**
3. Choose a scenario:
   - **Blocked — secrets + open route** (`default`) → expected: `BLOCKED`
   - **Approved — clean build** (`approved`) → expected: `ALLOW`
4. Click **Run demo** — view outcome, proof, and data sources

## API

```bash
curl -X POST http://localhost:8000/case-studies/cs-05/run \
  -H "Content-Type: application/json" \
  -d '{"scenario": "default"}'
```
