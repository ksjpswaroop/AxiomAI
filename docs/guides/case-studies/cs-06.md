# CS-06 User Guide — Procurement Agent Governance

**Tier:** 2 · **ID:** `cs-06`

## Problem
AI procurement agents may commit spend without approvals.

## Demo story
$45K GPU contract with new vendor triggers finance, legal, and security review.

## How to run (no code)

1. Open **Case Study Gallery** in the investor console
2. Select **Procurement Agent Governance**
3. Choose a scenario:
   - **Blocked — high spend new vendor** (`default`) → expected: `DENY`
   - **Escalate — manager approval** (`escalate`) → expected: `ESCALATE`
4. Click **Run demo** — view outcome, proof, and data sources

## API

```bash
curl -X POST http://localhost:8000/case-studies/cs-06/run \
  -H "Content-Type: application/json" \
  -d '{"scenario": "default"}'
```
