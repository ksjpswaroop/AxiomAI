# CS-02 User Guide — SOC2 Compliance Automation

**Tier:** 1 · **ID:** `cs-02`

## Problem
Manual SOC2 audits take weeks and miss configuration drift.

## Demo story
Azure AD and AWS Config evidence ingested; 8 controls evaluated automatically.

## How to run (no code)

1. Open **Case Study Gallery** in the investor console
2. Select **SOC2 Compliance Automation**
3. Choose a scenario:
   - **Full audit run** (`default`) → expected: `AUDIT_COMPLETE`
4. Click **Run demo** — view outcome, proof, and data sources

## API

```bash
curl -X POST http://localhost:8000/case-studies/cs-02/run \
  -H "Content-Type: application/json" \
  -d '{"scenario": "default"}'
```
