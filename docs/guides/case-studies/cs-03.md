# CS-03 User Guide — AI Customer Support Governance

**Tier:** 1 · **ID:** `cs-03`

## Problem
LLM support agents may approve refunds outside policy.

## Demo story
Agent proposes $350 refund 43 days post-purchase without return — DENIED with proof.

## How to run (no code)

1. Open **Case Study Gallery** in the investor console
2. Select **AI Customer Support Governance**
3. Choose a scenario:
   - **Deny — outside 30-day window** (`deny_outside_window`) → expected: `DENY`
   - **Deny — product not returned** (`deny_not_returned`) → expected: `DENY`
   - **Allow — compliant refund** (`allow_compliant`) → expected: `ALLOW`
   - **Escalate — mid-range amount** (`escalate_mid_amount`) → expected: `ESCALATE`
   - **Deny — duplicate refund** (`deny_already_refunded`) → expected: `DENY`
4. Click **Run demo** — view outcome, proof, and data sources

## API

```bash
curl -X POST http://localhost:8000/case-studies/cs-03/run \
  -H "Content-Type: application/json" \
  -d '{"scenario": "default"}'
```
