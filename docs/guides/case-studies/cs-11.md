# CS-11 User Guide — HR Policy Engine

**Tier:** 4 · **ID:** `cs-11`

## Problem
HR teams manually check leave policy eligibility.

## Demo story
Employee eligible for parental leave after tenure and hours requirements met.

## How to run (no code)

1. Open **Case Study Gallery** in the investor console
2. Select **HR Policy Engine**
3. Choose a scenario:
   - **Parental leave eligible** (`default`) → expected: `ALLOW`
   - **Denied — insufficient tenure** (`denied`) → expected: `DENY`
4. Click **Run demo** — view outcome, proof, and data sources

## API

```bash
curl -X POST http://localhost:8000/case-studies/cs-11/run \
  -H "Content-Type: application/json" \
  -d '{"scenario": "default"}'
```
