# CS-14 User Guide — Manufacturing QC

**Tier:** 4 · **ID:** `cs-14`

## Problem
Quality team manually traces defect patterns on assembly lines.

## Demo story
Misalignment and elevated scrap rate traced to feeder calibration.

## How to run (no code)

1. Open **Case Study Gallery** in the investor console
2. Select **Manufacturing QC**
3. Choose a scenario:
   - **Feeder calibration root cause** (`default`) → expected: `DIAGNOSIS_FOUND`
4. Click **Run demo** — view outcome, proof, and data sources

## API

```bash
curl -X POST http://localhost:8000/case-studies/cs-14/run \
  -H "Content-Type: application/json" \
  -d '{"scenario": "default"}'
```
