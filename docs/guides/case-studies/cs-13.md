# CS-13 User Guide — Cloud Cost Governance

**Tier:** 2 · **ID:** `cs-13`

## Problem
Engineering provisions expensive GPU clusters without FinOps review.

## Demo story
$45K/month GPU in new region with long commitment blocked.

## How to run (no code)

1. Open **Case Study Gallery** in the investor console
2. Select **Cloud Cost Governance**
3. Choose a scenario:
   - **Blocked — GPU over budget** (`default`) → expected: `DENY`
4. Click **Run demo** — view outcome, proof, and data sources

## API

```bash
curl -X POST http://localhost:8000/case-studies/cs-13/run \
  -H "Content-Type: application/json" \
  -d '{"scenario": "default"}'
```
