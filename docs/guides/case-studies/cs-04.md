# CS-04 User Guide — Healthcare Prior Authorization

**Tier:** 3 · **ID:** `cs-04`

## Problem
Manual prior auth reviews delay patient care.

## Demo story
Patient completed PT, imaging, and meets high-severity criteria for lumbar procedure.

## How to run (no code)

1. Open **Case Study Gallery** in the investor console
2. Select **Healthcare Prior Authorization**
3. Choose a scenario:
   - **Approved — criteria met** (`default`) → expected: `ALLOW`
   - **Denied — PT not completed** (`denied`) → expected: `DENY`
4. Click **Run demo** — view outcome, proof, and data sources

## API

```bash
curl -X POST http://localhost:8000/case-studies/cs-04/run \
  -H "Content-Type: application/json" \
  -d '{"scenario": "default"}'
```
