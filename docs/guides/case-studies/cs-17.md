# CS-17 User Guide — Sales Qualification

**Tier:** 4 · **ID:** `cs-17`

## Problem
SDRs manually score leads against ICP criteria.

## Demo story
250-seat security buyer routed to Senior AE per ICP rules.

## How to run (no code)

1. Open **Case Study Gallery** in the investor console
2. Select **Sales Qualification**
3. Choose a scenario:
   - **Route to Senior AE** (`default`) → expected: `ROUTED`
4. Click **Run demo** — view outcome, proof, and data sources

## API

```bash
curl -X POST http://localhost:8000/case-studies/cs-17/run \
  -H "Content-Type: application/json" \
  -d '{"scenario": "default"}'
```
