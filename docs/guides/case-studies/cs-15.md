# CS-15 User Guide — Education Degree Audit

**Tier:** 4 · **ID:** `cs-15`

## Problem
Registrar manually audits degree completion requirements.

## Demo story
Student met credit hours and GPA for graduation eligibility.

## How to run (no code)

1. Open **Case Study Gallery** in the investor console
2. Select **Education Degree Audit**
3. Choose a scenario:
   - **Graduation eligible** (`default`) → expected: `ALLOW`
   - **Insufficient credits** (`denied`) → expected: `DENY`
4. Click **Run demo** — view outcome, proof, and data sources

## API

```bash
curl -X POST http://localhost:8000/case-studies/cs-15/run \
  -H "Content-Type: application/json" \
  -d '{"scenario": "default"}'
```
