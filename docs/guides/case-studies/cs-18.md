# CS-18 User Guide — Agentic Trading Guardrail

**Tier:** 5 · **ID:** `cs-18`

## Problem
Autonomous trading agents may exceed risk limits.

## Demo story
Oversized tech position breaches sector concentration limit.

## How to run (no code)

1. Open **Case Study Gallery** in the investor console
2. Select **Agentic Trading Guardrail**
3. Choose a scenario:
   - **Blocked — position limit** (`default`) → expected: `BLOCKED`
   - **Approved trade** (`approved`) → expected: `ALLOW`
4. Click **Run demo** — view outcome, proof, and data sources

## API

```bash
curl -X POST http://localhost:8000/case-studies/cs-18/run \
  -H "Content-Type: application/json" \
  -d '{"scenario": "default"}'
```
