# CS-16 User Guide — Immigration Assistant

**Tier:** 4 · **ID:** `cs-16`

## Problem
Immigration paralegals manually verify H-1B eligibility.

## Demo story
Beneficiary meets specialty occupation and employer requirements.

## How to run (no code)

1. Open **Case Study Gallery** in the investor console
2. Select **Immigration Assistant**
3. Choose a scenario:
   - **Eligible to file** (`default`) → expected: `ALLOW`
   - **Cap exhausted** (`denied`) → expected: `DENY`
4. Click **Run demo** — view outcome, proof, and data sources

## API

```bash
curl -X POST http://localhost:8000/case-studies/cs-16/run \
  -H "Content-Type: application/json" \
  -d '{"scenario": "default"}'
```

## Data sources
