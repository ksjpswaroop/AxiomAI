# CS-01 User Guide — MSP Network Operations

**Tier:** 2 · **ID:** `cs-01`

## Problem
Client VPN users cannot connect; L1-L3 escalation takes hours.

## Demo story
Auth failures spiked after internal CA certificate expired on the VPN gateway.

## How to run (no code)

1. Open **Case Study Gallery** in the investor console
2. Select **MSP Network Operations**
3. Choose a scenario:
   - **Certificate expired** (`default`) → expected: `ROOT_CAUSE_FOUND`
   - **DNS failure (alternate)** (`dns_path`) → expected: `ROOT_CAUSE_FOUND`
4. Click **Run demo** — view outcome, proof, and data sources

## API

```bash
curl -X POST http://localhost:8000/case-studies/cs-01/run \
  -H "Content-Type: application/json" \
  -d '{"scenario": "default"}'
```
