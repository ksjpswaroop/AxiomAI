# CS-10 User Guide — Banking Loan Eligibility

**Tier:** 3 · **ID:** `cs-10`

## Problem
Loan underwriting requires consistent rule application.

## Demo story
Prime applicant: 780 FICO, low DTI, stable employment.

## How to run (no code)

1. Open **Case Study Gallery** in the investor console
2. Select **Banking Loan Eligibility**
3. Choose a scenario:
   - **Prime approval** (`default`) → expected: `ALLOW`
   - **Declined — low FICO** (`declined`) → expected: `DENY`
4. Click **Run demo** — view outcome, proof, and data sources

## API

```bash
curl -X POST http://localhost:8000/case-studies/cs-10/run \
  -H "Content-Type: application/json" \
  -d '{"scenario": "default"}'
```

## Data sources

- [FFIEC underwriting guidelines](https://www.ffiec.gov/) — Public domain / synthetic demo
- [FRED economic data (patterns)](https://fred.stlouisfed.org/) — Public domain / synthetic demo
