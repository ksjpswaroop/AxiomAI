# CS-07 User Guide — Cybersecurity Root Cause Analysis

**Tier:** 1 · **ID:** `cs-07`

## Problem
SOC analysts spend hours correlating SIEM alerts manually.

## Demo story
15-fact ransomware incident on Host42 — attack chain confirmed in minutes.

## How to run (no code)

1. Open **Case Study Gallery** in the investor console
2. Select **Cybersecurity Root Cause Analysis**
3. Choose a scenario:
   - **Ransomware on Host42** (`default`) → expected: `CONFIRMED`
4. Click **Run demo** — view outcome, proof, and data sources

## API

```bash
curl -X POST http://localhost:8000/case-studies/cs-07/run \
  -H "Content-Type: application/json" \
  -d '{"scenario": "default"}'
```
